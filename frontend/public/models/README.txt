Place face-api model files in this folder for offline use.

Recommended files (from @vladmandic/face-api):
- ssd_mobilenetv1_model-weights_manifest.json + *.bin
- face_landmark_68_model-weights_manifest.json + *.bin
- face_recognition_model-weights_manifest.json + *.bin

Builds will load models from /models by default.
